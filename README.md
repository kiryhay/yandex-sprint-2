# yandex-sprint-2

# Автодополнение текстов: LSTM vs DistilGPT2

Проект по сравнению лёгкой LSTM и предобученного трансформера для задачи автодополнения на мобильных устройствах.

## Запуск

1. Установить зависимости:
```bash
   pip install -r requirements.txt
```

2. Запустить ноутбук
```bash
jupyter notebook solution.ipynb
```


## Итоги

LSTM — ROUGE-1: 0.084, ROUGE-2: 0.016
DistilGPT2 — ROUGE-1: 0.028, ROUGE-2: 0.004

Вывод:
LSTM лучше, т.к. обучена на конкрутных данных, а DistilGPT2 обучена на слишком большом количестве данных
LSTM можно использовать для проектов на мобильных устройствах, где память сильно ограничена
